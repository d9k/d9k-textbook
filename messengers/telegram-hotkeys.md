# Telegram: hotkeys

- Back: `Esc`
- File to send select: `Ctrl + O`
- Folder switch: `Ctrl + Shift + Up/Down`,  `Ctrl + 1..9`
- Formatting of text remove: `Ctrl + Shift + N`
- Media edit (?): `Ctrl + E`
- Message to reply change: `Ctrl + Up/Down`
- Search in chat: `Ctrl + F`
