# Ionic vue

## Building project

```
ionic capacitor add
ionic capacitor sync
ionic capacitor copy
ionic capacitor run
```

```
npx cap add android
npx cap sync
npx cap copy
npx cap run
```

Since you're using the Vue project type, you must provide the ionic:build npm script so the Ionic CLI can build your project.
