Утилиты для grub
================

Обновить grub
-------------

	sudo update-grub

Графический редактор grub
-------------------------

	sudo add-apt-repository ppa:adabbas/1stppa
	sudo apt-get update
	sudo apt-get install grub-customizer
	grub-customizer &
