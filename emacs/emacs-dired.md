Emacs: режим dired
==================

[мануал](https://www.gnu.org/software/emacs/manual/html_node/emacs/Dired.html)

Переход в режим
~~~~~~~~~~~~~~~
ctrl+x d

Горячие клавиши
~~~~~~~~~~~~~~~
m / u - отметить / снять отметку
x - удалить выделенные
+ - создать папку
ctrl+x ctrl+f, ввести имя, enter - создать файл

Обновить
--------

`g`
