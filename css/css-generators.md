# CSS generators

## CSS color change filter generator

https://codepen.io/sosuke/pen/Pjoqqp

Example:

```
.to-theme-background-color {
  /* #000 -> #00af93 (persian green) */
  filter: invert(41%) sepia(95%) saturate(846%) hue-rotate(135deg) brightness(95%) contrast(104%);
}
```
