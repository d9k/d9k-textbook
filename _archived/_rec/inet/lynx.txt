Lynx Browser 2.8.5rel.1
==========
1. Установка.
2. Настройка.  
Открыть lynx.cfg в любом текстовом редакторе:
 
Для кириллизации
CHARACTER_SET:cp866 - кодировка экранных шрифтов (вывода на экран)
ASSUME_CHARSET:windows-1251 - кодировка html-документов
PREFERRED_CHARSET:windows-1251 - кодировка для запросов серверам при открытии страницы
 
Для контроля открывания страниц
PARTIAL:TRUE - показывать частично скачанный документ во время закачки
SHOW_KB_RATE:TRUE - показывать скорост скачивания в kb
 
Кэширование в памяти для 30 страниц
DEFAULT_CACHE_SIZE:30  
SOURCE_CACHE:MEMORY
 
Использование мыши для выбора ссылок (по желанию)
USE_MOUSE:TRUE
 
Просмотр изображений в консоли  
XLOADIMAGE_COMMAND:gl.exe %s
VIEWER:image/png:start gl.exe %s &:NON_XWINDOWS
VIEWER:image/gif:start gl.exe %s &:NON_XWINDOWS
VIEWER:image/bmp:start gl.exe %s &:NON_XWINDOWS
VIEWER:image/tif:start gl.exe %s &:NON_XWINDOWS
VIEWER:image/tiff:start gl.exe %s &:NON_XWINDOWS
VIEWER:image/jpeg:start gl.exe %s &:NON_XWINDOWS
VIEWER:image/jpg:start gl.exe %s &:NON_XWINDOWS
 
Просмотр изображений работает по нажатию клавиши "*" и в появляющемся списке ссылок на изображения выбрать нужный, во внешнем окне загружается выбранный рисунок.
Вьювер брал как плагин для ФАРа, скачал в свое время откуда-то.
Но, возможно, подойдет любой внешний просмотрщик, просто необходимо прописать пути.
 
Использование proxy (по необходимости)
Настройки можно взять из IE, Оперы
http_proxy:http://some.server.dom:port/

По желанию можно изменить цветовую схему.  
Для этого в текстовом редакторе открываем lynx.bat и прописываем путь к файлам *.lss, лежат в корне программы.
set lynx_lss=%HOME%\*.lss
По умолчанию установлен opaque.lss  
 
Основные горячие клавиши программы:
Q - выход из прогаммы
Ctrl-R - перезагрузить страницу
Ctrl-A - перейти на начало страницы
Top/Bottom - идти на предыдущую/следующую ссылку
g- перейти на требуемый URL
G- редактировать URL текущего документа и идти по нему
H- вызов справки
Z- прервать передачу данных по сети
m - выйти в основное меню
o - открытть окно опций программы
/ - поиск внутри докумена
n - поиск следующего включения
e- редактировать текущий документ
p - показать насторйки печати
a - добавить документ в букмарки
v - показать файл букмарок
V - показать список ссылок, посещенных в течение текущего сеанса работы с Lynx
d - скачать файл по текущей ссылке
k - показать список горячих клавиш Lynx
l - показать спискок ссылок текущего документа
* - показать/скрыть ссылки на картинки в текущем документе
 
Навигация:  
Page Up/Page Down - переход на предыдущуюю, следующую страницу
Left/Right - переход на предыдущий, следующий документ
 
Прокрутка:  
+ - Следующая страница документа (Page-Down)  
- - Предыдущая страница документа (Page-Up)  
SPACE- Следующая страница документа (Page-Down)  
b - Предыдущая страница документа (Page-Up)  
CTRL-A - Переход к первой сранице данного документа (Home)  
CTRL-E - Переход к последней странице данного документа (End)  
CTRL-B - Перейти к предыдущей странице (Page-Up)  
CTRL-F - Перейти к следующей странице (Page-Down)  
CTRL-N - Прокрутить текущий документ на две строки вперёд  
CTRL-P - Прокрутить текущий документ на две строки назад  
) - Прокрутить текущий документ на полстраницы вперёд  
( - Прокрутить текущий документ на полстраницы назад
