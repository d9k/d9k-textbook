FireFox 4 + Windows 7 = Люто тормозит
В первую очередь поможет плагин.

Memory Fox
https://addons.mozilla.org/ru/firefox/addon/53880/

далее что нашел по ускорению:

Для начала сделаем резервную копию about:config:
Для Ubuntu:
cp -R ~/.mozilla ~/.mozilla-backup
Для Windows XP:
Documents and Settings\\Application Data\Mozilla\Firefox\Profiles\ .default\
Для Windows Vista:
Users\AppData\Roaming\Mozilla\Firefox\Profiles\ .default\
Ускорим Firefox.

1. Начнем с удаления плагина для Skype: если ВКонтакте или Gmail начали тормозить, а вы недавно обновились до Skype 4.1, установив плагин для Firefox, попробуйте для начала удалить этот самый плагин, не удаляя саму программу Skype. Должно помочь. Надеюсь, что Skype скоро выпустит обновление для своего дополнения, потому что читая и слыша о скорости работы Firefox пользователи начнут переходить на другие браузеры.

2. Проверим сколько памяти использует Firefox: вызовем Диспетчер задач с помощью Ctrl-Alt-Del в Windows XP или Crtl-Alt-Esc в Vista и перейдем на закладку «Процессы». Найдем процесс, который называется firefox.exe и проверим, сколько памяти он использует, например 250,000 К – это 250 Мб. Попробуем уменьшить количество открытых вкладок и удалить плагины, которые толком не используем, а потом еще раз обратим внимание на использование памяти. Вероятно, что у Вас (как и у меня) среди установленных плагинов есть те, что захотелось проверить или посмотреть, но в дальнейшем Вы уже ими не пользуетесь или пользуетесь крайне редко.

3. Немного про процедуры обслуживания. Попробуем использовать программы бесплатно предоставляемые их разработчиками, для обслуживания нашего компьютера и самого браузера Firefox CCleaner (для Windows) или BleachBit (Windows или Linux) для того, чтобы регулярно чистить временные файлы браузера, кэш страниц и историю посещений.

4. Изменим конфигурационные файлы Firefox: ниже несколько способов, заключающихся в изменении конфигурации Firefox. Вам, наверное, приходилось сталкиваться с заметками о них на различных сетевых ресурсах.Изменить конфигурацию совсем несложно. Наберем about:config в адресной строке Firefox и нажмем Enter. Затем наберем network.http в поле Фильтр. Этим мы отберем только те строки, в названиях которых встречается словосочетание network.http. Теперь, собственно внесем необходимые изменения:

Нажмем два раза левой кнопкной мыши на строку ‘network.http.pipelining’ тем самым установив ее значение в true (само двойное нажатие и изменит значение с false на true).
даблклик на строку network.http.pipelining.maxrequests’ , затем в диалоговом окне введем значение, большее, чем 4, что установлено по умолчанию. В основном рекомендуется установить его в 8.
двойным нажатием левой кнопки мыши на ‘network.http.proxy.pipelining’ поменяем значение на true.
еще раз наложим фильтр на список, введя ‘browser.sessionstore.interval‘ (без кавычек), нажмем два раза левой кнопкой мыши и поменяем значение с 10000 (10 секунд) до 120000 (12 минут), тем самым изменив, как часто Firefox будет создавать точки восстановления Ваших сессий с 10 секунд до двух минут.
нажмем правой кнопкой мыши на страницу, выберем Создать->Целое. Установим название nglayout.initialpaint.delay, затем нажмем Ок. Присвоим этому параметру значение 0 (ноль) и еще раз нажмем Ок.

5. Отключим Firebug: если вы пользуетесь Firebug, то это вероятная причина медленной работы Firefox.

6. Оптимизируем базу данных SQL у Firefox: Firefox использует SQLite для хранения своих Ускорение Firefox с помощью SpeedyFoxмногочисленных настроек. Со временем база данных разрастается, а сам браузер замедляет свою работу. Рекомендую попробовать дополнение SpeedyFox, приложение, которое сделают базу данных более компактной без потери каких-либо данных. К сожалению, SpeedyFox в данный момент работает только под Windows.

7. Если ничего из вышеперечисленного не помогло, попробуйте обратиться на русский форум Mozilla.

8. Все равно недовольны? Попробуйте Google Chrome – достойную альтернативу Mozilla Firefox.
