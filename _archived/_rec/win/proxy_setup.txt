windows way

Иногда возникает необходимость обновлять антивирус через proxy. В случае, если прокси-сервер не требует авторизации — есть решение.

Метод:
1. Запуск cmd (командная строка) от имени администратора.
2. В консоли вводите команду — netsh winhttp import proxy ie. [Для Windows XP попробуйте также команду proxycfg -u]
3. Запускаете обновления в MSE.
Всё.

Этот самый прокси должен быть вначале прописан в настройках IE.
